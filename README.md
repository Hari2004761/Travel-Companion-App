# TravelMeet

**Find your travel companion.**

TravelMeet is an Android app that helps travelers connect with like-minded companions who share their destinations, travel dates, and interests.

## What it does

Users sign in with email/password or Google, then build a profile with their personal details, travel destinations, planned travel dates, and hobbies (hiking, photography, food, nightlife, etc.). The app uses this information to match travelers heading to the same places at the same time.

## Features

- **Authentication** — Email/password and Google Sign-In via Firebase
- **Profile setup** — Name, date of birth, gender, country of origin, and a profile photo
- **Travel preferences** — Add multiple destinations as chips, set a start and end date
- **Interest tags** — Select hobbies to help find compatible companions
- **Location support** — GPS-enabled for location-aware matching

## Tech stack

- Android (Java) with Material Design 3
- Firebase Authentication
- Google Sign-In SDK
